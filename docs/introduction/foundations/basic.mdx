| Component         | Description                                                                                                                                                                                                                                                                                                                                                                                                           |
| ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
| Entity            | Entity represents the user, and all accounts that belong to a specific user will have the same entity ID. &nbsp; [Learn More...](./components/entity/entity-guide) 
| Accounts          | Connections represent the link between your app and third-party platform accounts of your user. &nbsp; [Learn More...](./components/entity/entity-guide)
| Tools             | Tools refer to the various applications or tools that can be integrated and used by your agents via Composio. &nbsp; [Learn More...](./components/tools/tool-guide)
| Actions           | Actions are tasks that can be performed by your agents. &nbsp; [Learn More...](./components/actions/action-guide)
| Triggers          | Triggers are predefined conditions that, when met, initiate webhooks to your agents. &nbsp; [Learn More...](./components/triggers/trigger-guide)
| Integrations      | Integrations are customizable configurations (settings) used while creating a new connected account. &nbsp; [Learn More...](./components/integrations/integration-guide)
